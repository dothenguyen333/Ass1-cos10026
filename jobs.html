Company Positions

On this page, you can learn more about our company positions so that you can easily choose when to work in our office.

|1. Network Administrator|

A Network Administrator is responsible for designing, implementing, and maintaining an organisation's
computer networks. They ensure that data communication networks, including local area networks
(LANs), wide area networks (WANs), and intranet systems, function efficiently and securely.

They configure hardware such as routers, switches, and firewalls, and monitor network traffic to prevent
downtime. In case of network failures or cyber threats, they troubleshoot issues and implement security
measures to protect sensitive data.

Key Responsibilities:

  1. Installing, configuring, and maintaining network devices (routers, switches, access points).
  2. Monitoring network performance and troubleshooting connectivity issues.
  3. Ensuring network security through firewalls, VPNs, and encryption protocols.
  4. Managing network backup and disaster recovery plans.
  5. Updating network hardware and software to optimise performance.
  6. Supporting end-users with connectivity issues and providing documentation.

Essential Skills & Tools:

  • Knowledge of networking protocols (TCP/IP, DHCP, DNS, HTTP, FTP).
  • Experience with Cisco, Juniper, or similar network hardware.
  • Proficiency in network monitoring tools (Wireshark, SolarWinds, Nagios).
  • Understanding of security best practices (firewalls, IDS/IPS).
  • Ability to configure VLANs, VPNs, and subnetting.

|2. System Administrator|

A System Administrator (SysAdmin) manages and maintains an organisation's IT infrastructure,
including servers, operating systems, and applications. Their primary responsibility is ensuring that all
systems run smoothly and securely.

SysAdmins are responsible for installing software, managing user accounts, configuring security settings,
and monitoring system performance. They also ensure system backups are conducted regularly to
prevent data loss.

Key Responsibilities:

  1. Installing, updating, and configuring operating systems (Windows Server, Linux, macOS).
  2. Managing user access, permissions, and security settings.
  3. Performing system backups and disaster recovery planning.
  4. Monitoring system performance and troubleshooting errors.
  5. Implementing security patches and updates to prevent cyber threats.
  6. Providing technical support to employees regarding system-related issues.

Essential Skills & Tools:

  • Expertise in operating systems (Windows Server, Linux, Unix).
  • Experience with virtualisation (VMware, Hyper-V, VirtualBox).
  • Familiarity with scripting languages (PowerShell, Bash).
  • Understanding of cloud services (AWS, Azure, Google Cloud).
  • Knowledge of security protocols and authentication mechanisms.

|3. Cloud Engineer|

A Cloud Engineer specialises in designing, implementing, and managing cloud computing systems for
businesses. They help organizations migrate their infrastructure to cloud-based environments such as
Amazon Web Services (AWS), Microsoft Azure, or Google Cloud Platform (GCP).

Cloud Engineers ensure that cloud services are secure, scalable, and cost-effective. They configure
virtual machines, databases, and storage solutions while optimising performance.

Key Responsibilities:

  1. Designing and implementing cloud infrastructure for businesses.
  2. Managing cloud storage, databases, and virtual machines.
  3. Configuring and monitoring cloud security to protect sensitive data.
  4. Automating deployment and scaling using Infrastructure as Code (Terraform, CloudFormation).
  5. Troubleshooting cloud-related issues and optimising performance.
  6. Implementing backup and disaster recovery strategies for cloud systems.

Essential Skills & Tools:

  • Expertise in cloud platforms (AWS, Azure, Google Cloud).
  • Knowledge of containerisation (Docker, Kubernetes).
  • Experience with CI/CD pipelines (Jenkins, GitHub Actions).
  • Familiarity with scripting languages (Python, Bash, PowerShell).
  • Understanding of cloud security and compliance (IAM, encryption).

|4. Cybersecurity Analyst|

A Cybersecurity Analyst is responsible for protecting an organisation's IT infrastructure, networks, and
sensitive data from cyber threats such as hacking, malware, and phishing attacks.

They continuously monitor systems for vulnerabilities, investigate security incidents, and implement
security measures to prevent data breaches. Cybersecurity Analysts also ensure compliance with security
policies and industry regulations.

Key Responsibilities:

  1. Monitoring networks for security breaches and responding to cyber threats.
  2. Conducting vulnerability assessments and penetration testing.
  3. Implementing security policies, firewalls, and encryption protocols.
  4. Investigating and analysing security incidents to prevent future attacks.
  5. Keeping up to date with the latest cybersecurity threats and trends.
  6. Ensuring compliance with data protection laws (GDPR, HIPAA).

Essential Skills & Tools:

  • Knowledge of cybersecurity frameworks (NIST, ISO 27001).
  • Experience with security tools (Wireshark, Metasploit, Snort).
  • Understanding of encryption, firewalls, and access controls.
  • Ability to analyse and interpret security logs.
  • Strong analytical and problem-solving skills.

|5. IT Support Specialist|

An IT Support Specialist provides technical assistance to employees and customers who experience
hardware or software issues. They troubleshoot problems, install updates, and guide users through
troubleshooting steps.

They work at help desks, responding to technical support tickets via phone, email, or in-person. Their
goal is to resolve IT-related problems quickly to minimise downtime and maintain productivity.

Key Responsibilities:

  1. Assisting users with hardware, software, and network-related issues.
  2. Diagnosing and troubleshooting computer malfunctions.
  3. Installing and updating software applications.
  4. Setting up new devices and configuring them for users.
  5. Maintaining an IT support ticketing system to track issues.
  6. Educating employees about IT security and best practices.

Essential Skills & Tools:

  • Strong knowledge of operating systems (Windows, macOS, Linux).
  • Experience with IT support tools (Remote Desktop, TeamViewer).
  • Familiarity with troubleshooting hardware components (RAM, HDD, GPU).
  • Good communication skills to explain technical concepts to non-technical users.
  • Ability to work under pressure and resolve issues efficiently.
